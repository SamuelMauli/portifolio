Rust LavaLamp

Rust LavaLamp es un proyecto educativo desarrollado en Rust, inspirado en LavaRand de Cloudflare. Utiliza imágenes aleatorias de la API Picsum.photos para generar entropía criptográfica. Cada usuario es identificado por un hash de su nombre, y la contraseña se encripta en base a una clave derivada de una imagen única. La interfaz de terminal permite crear, editar y autenticar usuarios. El proyecto utiliza SHA256 y se enfoca en prácticas seguras de autenticación. Su objetivo es enseñar conceptos criptográficos de forma práctica y accesible.
