Rust LavaLamp

Rust LavaLamp é um projeto educacional em Rust inspirado no LavaRand da Cloudflare, que usa imagens aleatórias da API Picsum.photos para gerar entropia criptográfica. Cada usuário é identificado por um hash do nome e tem a senha criptografada com base em uma chave derivada de uma imagem única. A interface em terminal permite criar, editar e autenticar usuários. O projeto usa hashing com SHA256 e foca em práticas de segurança e autenticação. Seu objetivo é ensinar conceitos de criptografia de forma prática e acessível.
